## Program:
```
#Program to find the inverse of a matrix.
#Developed by: Dhandeeswaran selvakumar
#RegisterNumber:23006838
import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
inverse=np.linalg.inv(A)
print(inverse)
```
